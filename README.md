# lmu-paddock-companion
The LMU Paddock Companion app for logging laptimes in LMU

Welcome to the LMU Paddock Companion! This lightweight background application connects directly to Le Mans Ultimate to automatically track your lap times, and car setups, syncing them straight to your LMU Paddock dashboard.


# INSTALLATION & SETUP
1. Extract the ZIP file: Always extract the contents of the .zip file to a folder on your computer before running the app. Do not run the .exe directly from inside the ZIP.
2. Run the Application: Double-click LMU Paddock Companion.exe.
3. Log In: Click "LOGIN WITH BROWSER". A web page will open to authenticate your account. Once successful, you can close the browser tab.
4. Start Driving: Boot up Le Mans Ultimate. The Companion will automatically detect the game and start logging your laps as soon as you hit the track!

# IMPORTANT NOTE: WINDOWS SMARTSCREEN & ANTIVIRUS

Because LMU Paddock is an indie project, our application does not yet have an expensive digital corporate signature. As a result, Windows or your Antivirus might flag the application as "Unknown" or block it from running.

If you see the blue "Windows protected your PC" (SmartScreen) popup:
1. Click on "More info".
2. Click on "Run anyway".

Antivirus False Positives:
The app is built using Python and packaged with PyInstaller. Occasionally, overly aggressive antivirus software flags PyInstaller executables as a "False Positive". If the .exe is removed or blocked by your antivirus, please add the file to your antivirus exclusions/whitelist. 

Rest assured, the application is completely safe, does not hook into game memory (we use the official REST API), and is 100% cheat-safe.

# SETTINGS
At the bottom of the Companion app, you will find a few useful settings:
* Start with Windows: Check this box if you want the Companion to start automatically when you boot up your PC.
* Minimize to Tray on Close: When checked, clicking the 'X' in the top right corner will not close the app, but hide it in your system tray (next to your Windows clock). You can right-click the tray icon to open or quit the app.
